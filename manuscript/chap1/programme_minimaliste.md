## Programme minimaliste

Comme nous l'avons déjà vu, le programme suivant est le minimum que l'on puisse saisir :

    int main(){
    }

C'est un programme qui ne fait pas grand chose. Il rentre dans la fonction "main()" que l'appelle "point d'entrée du programme" puis se termine (en retournant le code "**0**" au système pour dire que tout est ok).
